We are using a dataset from https://archive.ics.uci.edu/dataset/891/cdc+diabetes+health+indicators. The Diabetes Health Indicators Dataset contains healthcare statistics and lifestyle survey information about people in general along with their diagnosis of diabetes. The 35 features consist of some demographics, lab test results, and answers to survey questions for each patient. The target variable for classification is whether a patient has diabetes or is healthy.
